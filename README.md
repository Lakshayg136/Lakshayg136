### 📁 Featured Projects

Here are some of my top open-source projects:

#### 🖥️ [macOS Portfolio OS](https://github.com/Lakshayg136/personal-portfolio)
An immersive, interactive personal portfolio website simulating a realistic macOS desktop environment. Built with React, Tailwind CSS, and Framer Motion, it features a simulated Finder, a working zsh terminal, custom mini-games, and a serverless contact email gateway.
*   **Tech Stack**: React 18, Tailwind CSS, Vite, Framer Motion, EmailJS
*   **Demo**: [lakshaygupta.vercel.app](https://personal-portfolio-tau-hazel-19.vercel.app/)

#### 📈 [ActiveCircle Dashboard](https://github.com/Lakshayg136/DT) *(Renaming to ActiveCircle)*
A smart business networking and productivity dashboard template. Designed for entrepreneurs to streamline professional connections, scheduling, and personal organization.
*   **Tech Stack**: HTML5, Vanilla CSS3, JavaScript (ES6)
*   **Key Features**: Interactive contact manager carousel, dynamic task progress board, responsive scheduling calendar.

#### 📊 [Sales Data Analytics Dashboard](https://github.com/Lakshayg136/Python-Project)
An interactive end-to-end data analytics dashboard that processes retail transactions from the Superstore dataset. Calculates business metrics and generates detailed, styled visualizations to identify profitability and product performance.
*   **Tech Stack**: Python, Streamlit, Pandas, Matplotlib, Seaborn
*   **Key Features**: Real-time multi-dimensional filters, 5 key business metrics, automated profit/loss visual color coding.

#### ⏱️ [StudyFlow Tracker](https://github.com/Lakshayg136/studyflow)
A high-performance focus tracker and tasks dashboard helping students optimize their academic workflows. Features a customizable Pomodoro timer, study sessions logger, and interactive metrics dashboard.
*   **Tech Stack**: React 18, TypeScript, Tailwind CSS, Lucide Icons
*   **Key Features**: Live focus metrics, LocalStorage data persistence, automated productivity tips module.

#### 🏫 [OpenRooms Mobile App](https://github.com/Lakshayg136/Open-Rooms)
A cross-platform mobile application designed for college students to quickly check room occupancy, find collaboration spaces, and locate vacant lecture halls.
*   **Tech Stack**: Flutter, Dart
*   **Key Features**: Available/Unavailable rooms directory, intuitive map indicators, responsive layout.
